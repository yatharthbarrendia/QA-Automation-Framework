# QA-Automation-Framework
Selenium-based QA automation framework using Java and TestNG implementing Page Object Model (POM) for testing e-commerce workflows.

## 📌 Overview

This project is a Selenium-based automation testing framework built using Java and TestNG to test an e-commerce web application.

## 🛠️ Tech Stack

* Selenium WebDriver
* Java
* TestNG

## ✅ Test Scenarios

* Valid Login Test
* Invalid Login Test

## 🧱 Framework Design

* Page Object Model (POM)
* Reusable components

## ▶️ How to Run

1. Install Java & Maven
2. Add ChromeDriver
3. Run testng.xml
